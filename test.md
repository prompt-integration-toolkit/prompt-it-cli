# Prompt Integration Toolkit Context

Prompt Integration Toolkit, abbreviated as Prompt-it, is a CLI-based tool designed for storing, organizing, versioning, publishing, and reusing prompts.

The application works as a prompt registry where users can save prompts, organize them by author and name, and access them later through terminal commands or supported AI agent integrations.

Prompt Integration Toolkit uses a shared GitHub registry to store public prompt files and their metadata, while user accounts and authentication are managed separately through Supabase.

The main goal of Prompt Integration Toolkit is to make prompts reusable, versioned, easy to distribute, and accessible across different environments such as CLI tools, web applications, and AI agents.

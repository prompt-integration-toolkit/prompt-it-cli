**Prompt-it CLI** is the terminal interface for the Prompt-it ecosystem.

It provides a simple way to interact with Prompt-it directly from the command line, allowing users to manage prompt-related actions without needing a web interface.

---

## Installation

```bash
npm install -g prompt-it
```

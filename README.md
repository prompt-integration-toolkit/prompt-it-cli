**Prompt Integration Toolkit CLI**, abbreviated as **Prompt-it CLI**, is the terminal interface for the Prompt Integration Toolkit ecosystem.

It provides a simple way to interact with Prompt Integration Toolkit directly from the command line, allowing users to manage prompt-related actions without needing a web interface. Commands and other abbreviated references use `prompt-it`.

---

## Installation

```bash
npm install -g prompt-it
```
